# 脑梗患者康复与预防复发指南

**患者基本情况：**
- 70岁女性
- 动脉夹层手术史（3年前）
- 高血压病史
- 新发脑梗，保守治疗
- 有语塞症状

---

## 一、后续康复方案

### 1. 急性期康复（发病1-7天）

**依据**：《中国急性缺血性卒中诊治指南2023》《中国脑梗死急性期康复专家共识》

**核心原则**：
- 轻-中度卒中患者，发病24小时后可进行床边康复、早期离床康复训练（Ⅰ级推荐，A级证据）
- 康复应以循序渐进方式进行，必要时在监护条件下进行

**具体措施**：

| 项目 | 内容 | 注意事项 |
|------|------|----------|
| **良肢位摆放** | 每2小时更换体位，预防压疮和关节挛缩 | 需准备3-4个枕头辅助 |
| **被动关节活动** | 各关节被动运动，维持关节活动度 | 每日2-3次，每次15-20分钟 |
| **床上活动训练** | 床上翻身、桥式运动、床上坐起训练 | 循序渐进 |
| **吞咽功能评估** | 通过饮水试验评估 | 若有吞咽困难需调整饮食质地 |
| **生命体征监测** | 密切监测血压、心率、呼吸、意识状态 | 最初48-72小时内不宜积极降压 |

**注意事项**：
- 严重卒中患者需在生命体征稳定后（通常24-48小时）再开始康复
- 避免过度活动导致病情加重
- 老年患者需特别注意跌倒预防

---

### 2. 亚急性期康复（1-3个月）

此期是康复的黄金期，建议转入康复科或康复专科医院。

**物理康复**：

| 类型 | 内容 | 频率 |
|------|------|------|
| 肢体运动训练 | Bobath技术、任务特异性训练 | 每日45-90分钟 |
| 平衡与协调训练 | 坐位平衡→站立平衡→步行平衡 | 循序渐进 |
| 肌力训练 | 抗阻训练从低强度开始 | 每周2-3次 |

**语言康复（针对语塞症状）**：

| 方法 | 内容 | 频率 |
|------|------|------|
| Schuell刺激法 | 听觉、视觉、触觉多感官刺激 | 每日30-60分钟 |
| 言语表达训练 | 单音→单词→短句→长句 | 每日练习 |
| 交流促进训练 | 手势、图片卡片辅助交流 | 家庭成员参与 |

**认知康复**：
- 记忆训练：使用记忆笔记本、日程表辅助
- 注意力训练：数字排序、卡片分类等任务
- 执行功能训练：计划制定、任务分解练习

---

### 3. 恢复期康复（3-6个月）

**康复目标**：
- 提高日常生活自理能力
- 促进回归家庭和社会
- 巩固前期康复成果

**具体方案**：

| 项目 | 内容 |
|------|------|
| ADL训练 | 进食、穿衣、洗漱、如厕训练，使用辅助器具 |
| 社区康复过渡 | 从医院康复过渡到社区/家庭康复 |
| 家庭环境改造 | 移除障碍物、安装扶手、使用防滑垫 |

## 二、预防再次血栓

### 1. 长期用药方案

**核心药物**：

| 类别 | 药物 | 用法 | 证据级别 |
|------|------|------|----------|
| **抗血小板** | 阿司匹林 | 100mg/日，长期 | Ⅰ级推荐，A级证据 |
| | 氯吡格雷 | 75mg/日 | 备选或联合 |
| **他汀类** | 阿托伐他汀 | 20mg/日 | Ⅰ级推荐，A级证据 |
| **降压药** | 个体化选择 | - | 需根据血压调整 |

**特殊考虑（动脉夹层手术史）**：
- 需与神经科/血管外科医生沟通，评估夹层愈合情况
- 若夹层已稳定愈合，抗血小板治疗相对安全
- 若存在活动性夹层或复发风险，抗凝治疗可能更适合

---

### 2. 血压控制目标

**老年患者血压目标**：

| 年龄 | 目标血压 | 备注 |
|------|----------|------|
| **65-79岁** | <140/90 mmHg，如能耐受可降至<130/80 mmHg | Ⅰ级推荐 |
| **80岁以上** | <150/90 mmHg，如能耐受可降至<140/90 mmHg | Ⅱa级推荐 |

**本患者（70岁）目标**：<140/90 mmHg

---

### 3. 血脂管理

**LDL-C目标（卒中患者为极高危）**：

| 目标 | 数值 |
|------|------|
| **首要目标** | LDL-C <1.8 mmol/L |
| **次要目标** | LDL-C降低≥50% |

**监测频率**：
- 开始用药后4-6周复查血脂
- 达标后每3-6个月复查
- 同时监测肝功能、肌酶

---

### 4. 定期复查项目

| 复查项目 | 频率 | 内容 |
|----------|------|------|
| **血压监测** | 每日家庭自测 | 晨起、睡前各1次 |
| **血脂** | 每3-6个月 | LDL-C、TC、TG |
| **血糖** | 每3-6个月 | 空腹血糖、HbA1c |
| **肝肾功能** | 每3-6个月 | ALT、AST、Cr |
| **颈动脉超声** | 每6-12个月 | 评估动脉硬化进展 |
| **心脏评估** | 每年 | 心电图、必要时超声心动图 |
| **认知评估** | 每6个月 | MMSE或MoCA量表 |

## 三、日常生活注意事项

### 1. 饮食建议

**推荐**：地中海式饮食

| 类别 | 建议 |
|------|------|
| **盐摄入** | <5g/日（钠<2g） |
| **脂肪** | 低饱和脂肪，减少肥肉、油炸食品 |
| **蛋白质** | 鱼类、豆制品、瘦肉 |
| **膳食纤维** | 全谷物、蔬菜、水果 |
| **钾摄入** | 蔬菜、水果 |

**每日参考**：
- 蔬菜：300-500g
- 水果：200-350g
- 全谷物：占主食1/3以上
- 鱼类：每周至少2次

---

### 2. 运动建议

**血压控制良好患者的运动推荐**：

| 类型 | 内容 | 频率 |
|------|------|------|
| 有氧运动 | 快走、慢跑、游泳、太极拳 | 每周≥150分钟 |
| 抗阻运动 | 辅助性力量训练 | 每周2-3次 |
| 柔韧训练 | 拉伸、八段锦 | 配合进行 |

**注意事项**：
- 避免清晨血压高峰时段剧烈运动
- 建议上午10点后或下午进行运动
- 从低强度开始，循序渐进

---

### 3. 作息规律

| 项目 | 建议 |
|------|------|
| **睡眠时间** | 每晚7-8小时 |
| **起床方式** | 醒后躺2-3分钟再缓慢坐起 |
| **午休** | 可适当午休30-60分钟 |

---

### 4. 情绪管理

卒中后抑郁焦虑发生率高达30-50%，需重视心理管理。

**策略**：
- 心理支持：家人理解、耐心陪伴
- 社交活动：适当参与社交，避免长期独处
- 兴趣爱好：培养舒缓型爱好
- 专业干预：必要时寻求心理医生帮助

---

### 5. 季节变化应对

秋冬脑梗复发率比春夏高30%以上。

| 季节 | 注意事项 |
|------|----------|
| **秋冬换季** | 及时添衣保暖 |
| **早晚低温** | 外出注意保暖 |
| **夏季高温** | 注意补充水分 |
| **室内外温差** | 进出空调房间注意过渡 |

---

## 四、严格禁忌事项

### 1. 饮食禁忌

| 类别 | 具体禁忌 | 原因 |
|------|----------|------|
| **高盐食品** | 腌制食品、咸菜、咸鱼 | 升高血压 |
| **高脂食品** | 动物内脏、肥肉、油炸食品 | 升高血脂 |
| **高糖食品** | 糖饮料、精制甜点 | 升高血糖、血脂 |
| **加工食品** | 腊肉、火腿、罐头 | 高盐、高添加剂 |
| **酒精** | 限制饮酒，最好戒酒 | 升高血压、诱发房颤 |

---

### 2. 活动禁忌

| 禁忌 | 危险 |
|------|------|
| **突然用力** | 血压骤升，诱发卒中 |
| **剧烈运动** | 心血管风险 |
| **晨练过猛** | 血压骤升 |
| **长时间热水浴** | 血管扩张，脑缺血 |
| **久坐不动** | 血流缓慢，血栓风险 |

---

### 3. 药物禁忌

| 禁忌 | 说明 |
|------|------|
| **自行停药** | 绝对禁止停用抗血小板药、降压药、他汀药 |
| **自行减量** | 不可擅自调整药物剂量 |
| **遗漏服药** | 严格按时服药 |
| **药物冲突** | 阿司匹林避免与其他抗凝药混用（除非医生指导） |

---

### 4. 生活习惯禁忌

| 禁忌 | 说明 |
|------|------|
| **吸烟** | 必须戒烟！卒中风险增加2-3.5倍 |
| **过量饮酒** | 戒酒或严格限制 |
| **熬夜** | 保证充足睡眠 |
| **情绪激动** | 避免动怒、焦虑、大悲大喜 |
| **憋尿憋便** | 及时排解，避免用力 |
| **猛然起床** | 醒后躺2-3分钟再缓慢坐起 |

## 五、家属护理要点

### 1. 日常监测

**每日必做**：

| 项目 | 频率 | 目标范围 |
|------|------|----------|
| **血压** | 晨起、睡前各1次 | <140/90 mmHg |
| **心率** | 测血压时同时记录 | 60-80次/分钟 |
| **意识状态** | 全天关注 | 有变化立即就医 |
| **肢体活动** | 观察有无变化 | 乏力加重需就医 |
| **言语状态** | 观察变化 | 突然加重需就医 |

**建议**：建立健康日记本，每日记录血压、心率、用药情况。

---

### 2. 急救准备

**必备准备**：

| 项目 | 内容 |
|------|------|
| **急救电话** | 120急救号码贴在显眼位置 |
| **就近医院** | 了解最近有"卒中绿色通道"的医院 |
| **病历资料** | 准备好病历摘要、用药清单、过敏史 |

**FAST识别法**：
- **F（Face）**：面部是否对称？有无口角歪斜？
- **A（Arm）**：双侧肢体是否有力？
- **S（Speech）**：说话是否清晰？
- **T（Time）**：如有症状，立即拨打120！

**卒中复发征兆（立即就医）**：
- 突发面部歪斜
- 突发肢体无力或麻木
- 突发言语不清或理解困难
- 突发眩晕、行走不稳
- 突发视力障碍

---

### 3. 心理支持

卒中后焦虑抑郁发生率30-50%。

| 方法 | 内容 |
|------|------|
| **倾听与理解** | 耐心倾听，理解情绪波动 |
| **情感交流** | 经常沟通，增强归属感 |
| **正面鼓励** | 肯定康复进步，树立信心 |
| **陪伴参与** | 参与康复训练，给予支持 |
| **专业求助** | 发现抑郁焦虑时及时寻求心理医生 |

## 六、核心要点总结

| 要点 | 内容 |
|------|------|
| **康复要早** | 发病24小时后即可开始，前3个月是黄金期 |
| **用药要稳** | 抗血小板药、他汀类、降压药必须长期规律服用 |
| **血压要控** | 目标<140/90 mmHg，每日家庭自测 |
| **血脂要降** | LDL-C目标<1.8 mmol/L |
| **生活要改** | 低盐低脂饮食、戒烟戒酒、适度运动 |
| **禁忌要严** | 突然用力、剧烈运动、熬夜、情绪激动必须避免 |
| **监测要勤** | 每日血压心率、定期复查血脂血糖 |
| **急救要快** | 掌握FAST识别法，卒中复发立即拨打120 |
| **心理要护** | 关注患者情绪，给予持续心理支持 |

## 七、特别提醒（针对本患者）

鉴于患者有**动脉夹层手术史（3年前）**：

- 与血管外科医生评估夹层愈合情况
- 根据评估结果确定抗栓方案（抗血小板vs抗凝）
- 定期复查血管影像，监测夹层状态
- 避免可能增加血管压力的活动
