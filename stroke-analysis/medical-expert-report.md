# 脑梗保守治疗方案分析报告

**作者**: medical-expert (stroke-analysis team)
**日期**: 2026-04-06
**患者情况**: 突发脑梗，医生建议保守治疗而非溶栓

---

## 一、保守治疗的具体方案和药物

根据《中国急性缺血性卒中诊治指南2023》及《脑血管病防治指南（2024年版）》，保守治疗主要包括以下内容：

### 1. 抗血小板治疗

**核心药物**：
- **阿司匹林**: 100-300mg/日，首选单药治疗
- **氯吡格雷**: 75mg/日，作为阿司匹林替代或联合用药
- **双联抗血小板治疗（DAPT）**: 针对轻型卒中（NIHSS≤3）或高危TIA（ABCD2≥4），推荐阿司匹林+氯吡格雷短期（21-90天）联合治疗

**指南依据**：
- AHA/ASA 2021指南推荐：非心源性缺血卒中首选抗血小板而非抗凝
- CHANCE研究证明：轻型卒中患者早期（24小时内）启动阿司匹林+氯吡格雷21天可显著降低90天卒中复发风险

### 2. 他汀类药物治疗

**核心药物**：
- **阿托伐他汀**: 20-40mg/日
- **瑞舒伐他汀**: 10-20mg/日

**目标**: LDL-C目标值<1.8mmol/L或降低≥50%

**作用机制**：
- 稳定动脉粥样硬化斑块
- 改善内皮功能
- 抗炎、抗氧化作用
- 促进神经保护和神经修复

### 3. 神经保护药物

中国临床常用的神经保护药物：
- **依达拉奉右莰醇**: 自由基清除剂，具有神经保护作用
- **尤瑞克林**: 改善脑血流，促进神经功能恢复
- **丁苯酞**: 改善缺血区微循环，促进神经重塑

### 4. 血压管理

**急性期血压策略**：
- 发病24-48小时内：允许性高血压，SBP目标≤220/120mmHg
- 保守治疗患者：若血压>220/120mmHg需降压处理

**长期血压控制**：
- 目标SBP<140mmHg
- 常用药物：长效钙拮抗剂、ACEI/ARB类

### 5. 血糖管理

- 急性期血糖目标：7.8-10mmol/L
- 避免低血糖（<3.3mmol/L可加重脑损伤）

### 6. 一般支持治疗

- 维持水电解质平衡
- 防治并发症（感染、深静脉血栓、压疮）
- 早期康复介入（肢体功能、语言、认知训练）

## 二、保守治疗的预期效果

### 1. 神经功能恢复

保守治疗无法实现血管再通，但通过以下机制改善预后：

| 机制 | 预期效果 |
|------|----------|
| 抗血小板 | 防止血栓扩展，降低早期复发风险 |
| 他汀类药物 | 稳定斑块，长期降低卒中风险约20-30% |
| 神经保护药物 | 减轻缺血损伤，促进神经重塑 |
| 早期康复 | 促进神经功能代偿，改善生活质量 |

### 2. 预期预后范围

**关键影响因素**：
- **梗死体积**: 小梗死预后较好
- **梗死部位**: 皮层下、基底节区梗死较皮层梗死预后差
- **基线NIHSS评分**: ≤5分预后较好，≥25分预后较差
- **年龄**: 年轻患者恢复潜力更大

**预期mRS评分**（改良Rankin量表）：
- 轻型卒中（NIHSS≤5）：约60-70%患者可达mRS 0-2（预后良好）
- 中型卒中（NIHSS 6-15）：约40-50%可达mRS 0-2
- 重型卒中（NIHSS≥16）：预后不良风险显著增加

## 三、保守治疗的风险

### 1. 疾病进展风险

**主要风险**：
- **卒中进展**: 约10-30%患者可能出现早期神经功能恶化（END）
- **梗死扩大**: 侧支循环差的患者梗死可能扩大
- **出血转化**: 自发性出血转化风险约5-10%

### 2. 药物相关风险

| 药物 | 主要风险 |
|------|----------|
| 抗血小板药物 | 出血风险（消化道出血约1-2%）、颅内出血风险增加 |
| 他汀类药物 | 肝功能损害、肌肉毒性（横纹肌溶解罕见） |
| 神经保护药物 | 相对安全，少数过敏反应 |

### 3. 并发症风险

- **深静脉血栓/肺栓塞**: 卧床患者风险增加
- **吸入性肺炎**: 吞咽功能障碍患者
- **跌倒风险**: 运动功能障碍患者

### 4. 长期风险

- **复发风险**: 年复发率约5-10%（需规范二级预防）
- **认知功能障碍**: 卒中后痴呆风险增加

## 四、什么情况下保守治疗优于溶栓

### 1. 溶栓绝对禁忌证（必须保守治疗）

根据AHA/ASA 2024及ESO 2023指南，以下情况**绝对禁用溶栓**：

| 绝对禁忌证 | 说明 |
|------------|------|
| 颅内出血史 | 任何类型的颅内出血 |
| 近3个月内严重头部外伤/卒中 | 增加出血风险 |
| 活动性内出血 | 包括消化道出血21天内 |
| 未控制的高血压 | 血压>185/110mmHg且无法控制 |
| CT显示大面积梗死 | >1/3 MCA供血区低密度 |
| 凝血功能障碍 | INR>1.7、血小板<100,000 |
| 近期使用抗凝药物 | DOAC 48小时内、LMWH 24小时内 |
| 感染性心内膜炎 | 栓子可能含细菌 |
| 动脉穿刺（非可压迫部位）7天内 | 增加出血风险 |

### 2. 溶栓相对禁忌证（需个体化评估）

| 相对禁忌证 | 处理建议 |
|------------|----------|
| 轻型卒中（NIHSS≤3）非致残性 | 保守治疗可能更安全 |
| 发病时间不明（醒后卒中） | 需高级影像评估组织窗 |
| 年龄≥80岁 | 溶栓风险增加，需综合评估 |
| 近期大手术（14天内） | 评估出血风险 |
| 心肌梗死3个月内 | 权衡风险获益 |
| 癫痫发作后 | 需确认卒中诊断 |

### 3. 时间窗外

- **发病>4.5小时**: 静脉溶栓时间窗关闭
- **发病>6小时**: 若非大血管闭塞伴可挽救组织，保守治疗为主
- **醒后卒中**: 无高级影像评估组织窗时，保守治疗更安全

### 4. 轻型卒中决策

**关键争议点**：
- PRIMS研究显示：轻型致残性卒中溶栓获益
- 非致残性轻型卒中：抗血小板治疗可能更安全

**决策依据**：
- NIHSS评分≤3分但存在致残性症状（如行走障碍、严重失语）：可考虑溶栓
- NIHSS评分≤3分且症状轻微可接受：保守治疗+早期DAPT

## 五、保守治疗的预后评估

### 1. 常用评估工具

| 评估工具 | 内容 | 时间点 |
|----------|------|--------|
| **NIHSS评分** | 神经功能缺损程度 | 入院、24h、7d、90d |
| **mRS评分** | 功能结局 | 90天、1年 |
| **BI指数** | 日常生活能力 | 出院、90天 |
| **ASPECTS评分** | CT梗死范围评估 | 急性期 |

### 2. 预后预测因素

**良好预后预测因素**：
- 年龄<60岁
- 基线NIHSS≤5分
- ASPECTS评分≥8分
- 良好侧支循环
- 小梗死体积（<30ml）
- 早期康复介入

**不良预后预测因素**：
- 年龄≥80岁
- 基线NIHSS≥16分
- ASPECTS评分≤5分
- 大面积梗死（>70ml）
- 侧支循环差
- 合并糖尿病、心房颤动
- 早期神经功能恶化

### 3. 预后评估流程

```
入院评估 → NIHSS评分 → ASPECTS评分 → 制定治疗策略
    ↓
24小时复查 → NIHSS变化 → 判断是否END
    ↓
7天评估 → NIHSS评分 → 早期预后判断
    ↓
90天随访 → mRS评分 → 最终预后评估
```

### 4. mRS评分分级解读

| mRS评分 | 功能状态 | 临床意义 |
|---------|----------|----------|
| 0 | 完全无症状 | 完全康复 |
| 1 | 有症状但无残疾 | 轻度影响 |
| 2 | 轻度残疾 | 可独立生活 |
| 3 | 中度残疾 | 需部分帮助 |
| 4 | 重度残疾 | 需全面护理 |
| 5 | 严重残疾 | 卧床、失禁 |
| 6 | 死亡 | — |

**预后良好定义**: mRS 0-2分
**预后不良定义**: mRS 3-6分

## 六、总结与建议

### 保守治疗的定位

保守治疗并非"放弃治疗"，而是针对不适合溶栓患者的重要治疗策略。其核心目标是：

1. **预防卒中进展和复发**
2. **保护缺血半暗带组织**
3. **促进神经功能重塑**
4. **改善长期生活质量**

### 患者沟通要点

医生建议保守治疗通常基于以下原因：
- 存在溶栓禁忌证
- 发病时间超过溶栓窗
- 轻型卒中且非致残性
- 大面积梗死风险高
- 患者整体状况不适合溶栓

保守治疗仍有明确的获益，需积极配合治疗并尽早启动康复。
