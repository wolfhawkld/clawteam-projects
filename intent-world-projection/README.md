# Intent-World-Projection

## 研究问题

探索通过人类语言的抽象建模来实现 AI 意图理解的可能性。

### 核心问题

1. **语言作为世界模型** - 人类语言是否可以作为对世界和人类认知逻辑的有效建模，让 AI 以类似方式理解世界和意图？

2. **对话系统意图分析** - 如果第一条不可行，能否在对话式知识型 agent 系统中通过意图分析设计多层级多维度召回流程？

3. **文献支撑** - 寻找相关科研论文进行分析论证

4. **可行性评估** - 确保方案无重大逻辑漏洞，有实际价值

-  - 系统设计方案
-  - 验证方案和实验设计
-  - 最终输出报告
